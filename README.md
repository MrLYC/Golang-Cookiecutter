# Golang web template
[![Build Status](https://travis-ci.org/MrLYC/Golang-Cookiecutter.svg?branch=master)](https://travis-ci.org/MrLYC/Golang-Cookiecutter)

## Getting started

```console
$ git clone <Golang-Cookiecutter>
$ cookiecutter Golang-Cookiecutter  # type app name
$ cd <APP>
$ make update
$ make
$ ./bin/<APP>
```
